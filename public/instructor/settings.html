Course basics

Course level

Year level

The academic year this course targets. It is set automatically from the course number (e.g. "BIOC 401" → 4th Year) and is used to tailor responses to your students and to flag higher-level material in the Super Course. Change it here if the default is wrong.

Student chat

How the regular student chat retrieves and cites course material.

Retrieval & sources

Student chat - Top-K

Number of course-material chunks regular student chat retrieves before answering. Default is 3.

Additive retrieval

When enabled, student chat for a selected unit can also use earlier published units from the course. When disabled, chat uses only the selected unit.

Additional material secondary search

When enabled, additional materials are de-prioritized: student chat searches lecture notes and practice materials first, and only searches additional materials if nothing relevant is found there.

Enable source attribution downloads

When enabled, source citations in student chat responses become clickable download links to the underlying uploaded documents.

Base system prompt

Instructions for the inline chat that appears when students answer quiz questions incorrectly. The chat helps students understand why the correct answer is right.

Privacy & sessions

Privacy & session settings

Anonymize students

When enabled, student names are hidden throughout your instructor dashboard. The Student Hub remains available and still shows student names, but individual struggle topics are hidden there. Student counts and activity data remain visible. This setting only affects your own view.

Student idle timeout (minutes)

Time before a student is prompted to confirm they are still active. Default is 4 minutes. The student sees a timer pop up halfway through the timeout period.

Bucket settings

These settings are shared - edits apply to every instructor and student using the bucket.

Super Course bucket

Select a bucket to edit, or delete the selected bucket. Deleting a bucket removes it from every course and hides it from students.

Bucket name

Shown to students in the Super Course picker (e.g. "2nd Year Biochem").

Year level

Used to label and order the bucket for students. Does not control access (enrollment does).

Show students this bucket

When enabled, enrolled students can see and chat this bucket in the Super Course picker.

Advanced retrieval & prompt settings

Instructor Super Course Top-K

Number of chunks instructor Super Course chat retrieves before answering. Default is 8.

Student Super Course Top-K

Number of chunks student Super Course chat retrieves before answering. Default is 8.

Include inactive courses

When enabled, inactive opted-in courses can contribute material to Super Course chat. Off by default.

Include Super Chat notes

When enabled, shared instructor notes are blended into instructor Super Chat answers alongside course material. On by default. (Instructor chat only - students never see notes.)

Notes retrieval share

Fraction of the instructor Top-K chunks reserved for notes. Default 0.25 (e.g. with Top-K 8, ~2 chunks come from notes and ~6 from course material). Unused note slots fall back to course material.

Notes minimum similarity

Minimum similarity (0–1) a note must reach to be used in an answer. Default 0.25. Higher = stricter (fewer notes used); lower = more permissive. Tuned for the current embedding model; raise it if irrelevant notes appear.

Instructor Super Course prompt

System prompt used for instructor Super Course chat.

Student Super Course prompt

System prompt used for student Super Course chat.

Student answer-level modifiers

Appended to the student prompt based on the level the student picks in chat. Undergraduate is the default selection.

Instructor answer-depth modifiers

Appended to the instructor prompt based on the depth the instructor picks in chat. Standard is the default selection.

Course lifecycle

Course lifecycle

Deactivate course

When a course is deactivated, students and TAs are blocked from using it. Instructors can still open the course here and reactivate it later.

Active Students and TAs can currently use this course.

Transfer course

Create a brand-new course copy from the current course. Materials are recopied and re-parsed, TAs keep their permissions if selected, and no students, chats, or historical activity are carried over.

Per-unit transfer

Learning objectives and assessment questions start on by default. Documents also bring their parsed topics along to the new course.

Mental health detection settings

Configure the AI prompt used to silently detect mental health concerns in student conversations. The detection runs in parallel and does not affect the student's chat experience. Flags appear on the Flagged Content page.
